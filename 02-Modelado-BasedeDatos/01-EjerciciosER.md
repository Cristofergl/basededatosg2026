# Ejercicios del modelo E-R


## Ejercicio 1.


un hospital registra informacion de sus pacientes:


>De cada paciente se almacena:

 - numero de paciente que lo identifica
 - Nombre
 - Fecha de Nacimiento

 de cada expediente Medico se almacena

 - Numero de expediente
 - Fecha de apertura
 - Tipo de sangre
 >Reglas de negocio
 1. cada paciente debe tener exactamente un expediente medico
 2. cadae expediente medico pertenece a un unico paciente
 3. no puede exixtir un expediente sin paciente
 4. no puede existir un paciente si expediente

 > que se debe realizar
 - Identificar la Entidades
 - Identificar Atributos
 - Determinar la cardinalidad
 - Determinar laparticipacion sin expediente
